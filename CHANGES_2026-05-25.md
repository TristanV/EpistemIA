## 📝 Notes de Maintenance

### Si vous modifiez `schema.js`
→ Tous les fichiers thématiques doivent utiliser les types valides  
→ Relancer `npm run validate` après

### Si vous ajoutez une thématique
→ Importer dans `resources/index.js`  
→ Relancer `npm run validate`  
→ Ouvrir une PR

### Si vous trouvez une erreur
→ Ouvrir une GitHub Issue ou PR avec `fix:`  
→ Inclure la source de la correction
